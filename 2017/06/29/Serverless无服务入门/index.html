Serverless无服务入门

发表于 2017-06-29 | 分类于 学习资源 | |  阅读次数

亚马逊开放lambda FAAS，阿里也开始跟上了无服务。看这样子，平台功能越来越强大，一般开发者真的是只能专注于实现“业务”了。
